# 分析模型参考

本文档定义了7维度评分算法、权重参数、赔率解读公式和信心指数计算方法。

---

## 一、7维度评分模型

### 1.1 总览

| 维度 | 权重 | 数据来源 | 评分范围 |
|------|------|----------|----------|
| 球队实力 | 25% | FIFA排名 + 身价 + 阵容深度 | 0-100 |
| 近期状态 | 20% | 近10场胜率/进失球 | 0-100 |
| 历史交锋 | 15% | 近5-10次对阵记录 | 0-100 |
| 伤停影响 | 15% | 核心球员缺阵评估 | 0-100 |
| 主客场 | 10% | 主场优势系数 | 0-100 |
| 赔率信号 | 10% | 欧赔隐含概率 + 亚盘水位 | 0-100 |
| 其他因素 | 5% | 天气、裁判、赛程密度 | 0-100 |

**总分计算**：
```
总分 = Σ(维度得分 × 权重)
信心指数 = |主队总分 - 客队总分| / 100 × 100%
```

---

### 1.2 球队实力评分（25%）

#### FIFA排名评分

| 排名区间 | 得分 |
|----------|------|
| 1-5 | 100 |
| 6-10 | 90 |
| 11-15 | 80 |
| 16-20 | 70 |
| 21-30 | 60 |
| 31-40 | 50 |
| 41-50 | 40 |
| 51+ | 30 |

#### 球队身价评分

| 身价区间 | 得分 |
|----------|------|
| €1.5B+ | 100 |
| €1B - 1.5B | 90 |
| €700M - 1B | 80 |
| €500M - 700M | 70 |
| €300M - 500M | 60 |
| €200M - 300M | 50 |
| €100M - 200M | 40 |
| < €100M | 30 |

#### 阵容深度评分

```
阵容深度 = (主力11人平均评分 + 替补7人平均评分) / 2

主力评分来源：Sofascore球员评分、Transfermarkt身价
替补评分：身价/10 + 近期出场次数权重
```

#### 综合计算

```
球队实力 = FIFA排名得分 × 0.4 + 身价得分 × 0.35 + 阵容深度 × 0.25
```

---

### 1.3 近期状态评分（20%）

#### 近10场胜率

```
胜率得分 = 胜率 × 100
# 例如：近10场7胜2平1负，胜率70%，得分70
```

#### 进失球评分

```
场均进球得分 = min(场均进球 × 25, 50)  # 最高50分
场均失球得分 = max(50 - 场均失球 × 20, 0)  # 失球越多分越低

进失球综合 = 场均进球得分 + 场均失球得分
```

#### 对手强度调整

```
调整后得分 = 原始得分 × 对手强度系数

对手强度系数：
- 对阵排名前10：1.2
- 对阵排名11-20：1.1
- 对阵排名21-30：1.0
- 对阵排名31+：0.9
```

#### 综合计算

```
近期状态 = 胜率得分 × 0.5 + 进失球综合 × 0.3 + 对手强度调整 × 0.2
```

---

### 1.4 历史交锋评分（15%）

#### 近5次对阵

```
交锋得分 = (胜场数 × 20 + 平局数 × 10) / 5 × 100 / 100
# 例如：3胜1平1负，得分 = (3×20 + 1×10) / 5 × 100 = 100
```

#### 进球优势

```
进球优势 = (总进球 - 总失球) × 5
# 每多进1球加5分，最多加25分
```

#### 时间衰减

```
近期交锋权重更高：
- 最近1次：× 1.5
- 第2次：× 1.3
- 第3次：× 1.1
- 第4次：× 1.0
- 第5次：× 0.9
```

#### 综合计算

```
历史交锋 = 交锋得分 × 0.6 + 进球优势 × 0.4
# 最终结果限制在0-100范围内
```

---

### 1.5 伤停影响评分（15%）

#### 缺阵球员分类

| 类型 | 定义 | 扣分 |
|------|------|------|
| 核心球员 | 身价前5或评分前5 | -15分/人 |
| 主力球员 | 首发11人 | -10分/人 |
| 重要替补 | 出场时间前15 | -5分/人 |
| 普通替补 | 其他 | -2分/人 |

#### 位置影响系数

```
位置系数：
- 前锋缺阵：× 1.2（进攻影响大）
- 中场核心缺阵：× 1.3（组织影响大）
- 后防核心缺阵：× 1.1
- 门将缺阵：× 1.4（影响最大）
```

#### 综合计算

```
基础分 = 100
总扣分 = Σ(球员类型扣分 × 位置系数)
伤停影响 = max(基础分 - 总扣分, 0)
```

---

### 1.6 主客场评分（10%）

#### 主客场系数

| 情况 | 系数 | 基础分 |
|------|------|--------|
| 主场 | 1.15 | 115 |
| 中立场 | 1.0 | 100 |
| 客场 | 0.9 | 90 |

#### 主场优势调整

```
主场优势受以下因素影响：
- 球迷支持：+5分（世界杯通常满场）
- 旅行距离：每1000km -2分
- 气候适应：±5分
- 时差影响：每2小时 -3分
```

#### 综合计算

```
主客场 = 基础分 × 系数 + 调整分
# 最终结果限制在0-100范围内
```

---

### 1.7 赔率信号评分（10%）

#### 欧赔隐含概率

```
# 计算隐含概率（扣除利润率）
利润率 = 1/主胜赔率 + 1/平局赔率 + 1/客胜赔率 - 1
# 通常利润率在5%-8%之间

主胜隐含概率 = (1/主胜赔率) / (1 + 利润率)
平局隐含概率 = (1/平局赔率) / (1 + 利润率)
客胜隐含概率 = (1/客胜赔率) / (1 + 利润率)

赔率信号得分 = 隐含概率 × 100
```

#### 亚盘水位解读

```
主队水位 < 0.85：看好主队，得分 +10
水位 0.85-0.95：均衡，不调整
水位 > 0.95：看好客队，得分 -10

让球盘口：
- 让0.5球：实力接近
- 让1球：略有优势
- 让1.5球：明显优势
- 让2球+：绝对优势
```

#### 大小球信号

```
大2.5球赔率 < 1.8：预期进球多，进攻型比赛
大2.5球赔率 > 2.0：预期进球少，防守型比赛
```

#### 综合计算

```
赔率信号 = 欧赔隐含概率得分 × 0.7 + 亚盘水位调整 × 0.3
```

---

### 1.8 其他因素评分（5%）

#### 天气影响

```
极端天气（暴雨/大雪/高温）：-10分
普通天气：0分
理想天气：+5分
```

#### 裁判因素

```
严裁（出牌多）：防守型球队 +5分
松裁（出牌少）：进攻型球队 +5分
```

#### 赛程密度

```
近10天3场以上：-10分（疲劳）
正常赛程：0分
休息充足：+5分
```

## 二、信心指数计算

### 2.1 计算方法

```
主队总分 = Σ(各维度得分 × 权重)
客队总分 = Σ(各维度得分 × 权重)

差距 = |主队总分 - 客队总分|
信心指数 = min(差距 / 20 × 100%, 100%)
# 差距20分对应100%信心
```

### 2.2 信心等级

| 信心指数 | 等级 | 说明 |
|----------|------|------|
| > 80% | 强烈推荐 | 多维度高度一致 |
| 60-80% | 推荐 | 多数维度支持 |
| 40-60% | 谨慎推荐 | 存在分歧 |
| < 40% | 不建议 | 高度不确定 |

## 三、投注建议生成

### 3.1 建议类型

| 类型 | 条件 |
|------|------|
| 胜负平 | 信心指数 > 60% |
| 让球盘 | 强弱差距明显，让球合理 |
| 大小球 | 进失球数据支持 |
| 比分预测 | 历史交锋 + 近期状态综合 |
| 半全场 | 通常不推荐，除非数据极强 |

### 3.2 建议理由模板

```
推荐理由应包含：
1. 核心数据支撑（如"近10场7胜，状态出色"）
2. 关键因素（如"对手主力前锋伤缺"）
3. 赔率价值（如"赔率2.2有投注价值"）
4. 风险提示（如"历史交锋不占优，需谨慎"）
```

## 四、多场对比分析

### 4.1 对比维度

当用户需要比较多场比赛时，从以下维度对比：

| 维度 | 说明 |
|------|------|
| 确定性 | 信心指数对比 |
| 赔率价值 | 隐含概率 vs 实际判断 |
| 风险等级 | 冷门可能性 |
| 推荐优先级 | 综合评分排序 |

### 4.2 优先级排序

```
推荐优先级 = 信心指数 × 0.5 + 赔率价值 × 0.3 + 数据完整度 × 0.2

赔率价值 = |判断概率 - 隐含概率| × 100
# 正数表示有投注价值
```

## 五、特殊场景处理

### 5.1 淘汰赛 vs 小组赛

```
淘汰赛特点：
- 平局进入加时/点球
- 球队更加谨慎
- 小球概率更高

小组赛特点：
- 必须赢球出线时进攻积极
- 已出线可能轮换
- 大球概率较高
```

### 5.2 东道主优势

```
世界杯东道主通常：
- 实力评分 +10
- 主客场系数提升至1.25
- 球迷支持加成 +10
```

### 5.3 卫冕冠军

```
卫冕冠军特点：
- 关注度加成，压力也大
- 阵容稳定性高
- 对手更加重视
```

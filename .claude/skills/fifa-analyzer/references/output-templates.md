# 输出模板参考

本文档定义了14种输出模板，每种对应一种用户意图。所有输出必须使用中文，并在末尾附加免责声明。

## 通用格式规范

### 1. 国旗表示
使用 emoji 国旗：🇫🇷 法国、🇩🇪 德国、🇧🇷 巴西

### 2. 数字格式
- 金额：€1.5B（十亿）、€500M（百万）
- 百分比：70%（保留整数）
- 赔率：2.20（保留两位小数）
- 比分：2-1（主队在前）

### 3. 时间格式
- 日期：YYYY-MM-DD
- 时间：HH:MM（24小时制）
- 比赛时间：第35分钟 / 上半场补时2分钟

### 4. 推荐等级标识
- ⭐⭐⭐⭐⭐ 强烈推荐（信心>80%）
- ⭐⭐⭐⭐ 推荐（信心60-80%）
- ⭐⭐⭐ 谨慎推荐（信心40-60%）
- ⭐⭐ 观望（信心20-40%）
- ⭐ 不建议（信心<20%）
